Easily upload, re-order multiple featured images for your posts, pages and custom post types

== Description ==

This plugin allows you to easily add as many featured images as you want to your post. You can upload 1 or bulk images using the default WordPress Media Uploader. Drag & drop
to re-order your images and delete any with ease. Works with a shortcode or PHP function call for developers.
